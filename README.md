# Chat Bot

## Stack

- Next.js
- React
- TypeScript
- Material-UI
- styled-components

## Install dependencies

```
yarn install
```

## Development

### Serve & watch

```
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Testing

Run all tests (this command is also run pre-commit):

```
yarn test
```

## Build

```
yarn build
```
